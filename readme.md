## go 指针
> 	x := 1
> 	p := &x //代表某个变量的地址
> 	fmt.Println("值", *p)
> 	fmt.Println("地址&x=p=", p)
> 	*p = 2 //*xxx 是某个变量，可以直接对其进行操作（赋值）相当于变量
> 	 问：*xxx 和变量 x的区别是什么？？？
> 	每次我们对一个变量取地址，或者复制指针，我们都是为原变量创建了新的别名。例如，*p就是变量v的别名。
> 	指针特别有价值的地方在于我们可以不用名字而访问一个变量，

# go ch2
Go语言将数据类型分为四类：
**基础类型、复合类型、引用类型和接口类型。**
> 基础类型，包括：**数字、字符串和布尔型**。
> 复合数据类型——**数组**和**结构体**——是通过组合简单类型，来表达**更加复杂的数据结构**。
> 引用类型包括**指针、切片、字典、函数、通道**
